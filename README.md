# 📁 FileHub - Gestor de Documentos Médicos

Aplicación web para gestionar documentos médicos con Supabase.

## 🚀 Características

- ✅ Autenticación con email/contraseña
- ☁️ Almacenamiento en Supabase
- 📊 Base de datos PostgreSQL
- 🔐 Control de acceso por usuario

## 📋 Requisitos

- Cuenta Supabase activa
- GitHub token (para deployments)

## 🔧 Configuración

1. Instala dependencias:
   ```bash
   npm install
   ```

2. Copia `.env.local`:
   ```bash
   cp .env.example .env.local
   ```

3. Inicia el servidor:
   ```bash
   npm run dev
   ```

## 🌐 Deploy

### GitHub Pages
El sitio se deploya automáticamente a: `https://ramongalera22-ai.github.io/filehubdef`

### Cambios
Simplemente haz push a `main` o `master` y GitHub Actions se encargará del resto.
